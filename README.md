# MIDI to Psych Converter

A lightweight MIDI to Friday Night Funkin' Psych Engine converter written completely from scratch.

## Features

- No external libraries
- Custom MIDI parser
- Psych Engine JSON output
- Supports Player / Opponent / Both
- Optional JSON minification

## Requirements

Python 3.10+

Download:

https://www.python.org/downloads/

## Run

python miditochart.py
